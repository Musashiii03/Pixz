# Pixz v1.1.0

Photo and video gallery with modern UI and smart features.

## Installation

1. Download `Pixz-1.1.0-Portable.zip`
2. Extract and run `Pixz.exe`
3. No Java installation required

## What's New

- "No thumbnail generated" text instead of placeholder images
- Smart refresh: regenerates missing thumbnails and detects file changes
- Better video thumbnails (no more black frames)
- Scroll position preserved when viewing media
- Horizontally scrollable folder buttons
- Video rotation support (auto + manual with 'R' key)
- Faster thumbnail generation (8x parallelism)
- Fixed session persistence and shutdown issues

## Keyboard Shortcuts

- `Space` - Play/Pause
- `M` - Mute/Unmute
- `R` - Rotate
- `Arrows` - Navigate
- `F/F11` - Fullscreen
- `ESC` - Close

## Supported Formats

**Images**: JPG, PNG, GIF, BMP  
**Videos**: MP4 (H.264), AVI, MOV, M4V

Note: H.265/HEVC, VP9, WebM not supported. Use "Open in System Player" for these.
