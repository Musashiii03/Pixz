# Pixz

A modern photo and video gallery for Windows with smooth performance, smart media handling, and a clean, minimal UI.

---

## ✨ Features

* 🖼️ **Masonry Gallery Layout**
  Browse photos and videos in a responsive, visually balanced grid

* 🎬 **Unified Media Viewer**
  Consistent viewing experience for both images and videos

* 🔄 **Smart Media Rendering**
  Accurate scaling, rotation, and aspect ratio handling

* ▶️ **Video Playback Controls**
  Play, pause, loop, seek, and rotate videos seamlessly

* 📁 **Multiple Folder Support**
  Manage and switch between different media directories

* 🔍 **Search, Filter & Sort**
  Quickly find media using flexible filtering options

* ⚡ **Smart Refresh System**
  Automatically updates thumbnails and detects changes

* ⌨️ **Keyboard Shortcuts**
  Fast navigation and control without using the mouse

* 🌙 **Modern Dark Theme**
  Clean UI with improved contrast and visual hierarchy

* 💾 **Session Persistence**
  Remembers your last state and restores it on launch

## 📦 Installation

Download the latest portable release from [Releases](../../releases) and run:

```bash
Pixz.exe
```

No installation required.

**Videos**
